# roboflex.util.png

Roboflex support for png compression to file and memory, and png decompression from memory.

Useful for compressing images over slow transports (wifi, etc).

## System Dependencies

    None! We build lodepng from source...

## Nodes

    PNGCompressor
    PNGDecompressor
